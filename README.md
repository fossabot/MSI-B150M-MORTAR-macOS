使用Clover+GPT方式安装，已更新到最新的macOS High Sierra 10.13。
安装High Sierra时请使用config-to-install-10.13.plist文件。

### 主机配置：
| 硬件  | 型号 |
| ------------- | ------------- |
| 主板  | 微星B150M MORTAR  |
| CPU  | Intel i3-6100  |
| 显卡  | Intel HD530  |
| 声卡  | Realtek ALC892 |

### 已知bug：
* 声卡使用万能驱动，有轻微的电流声
* 睡眠后无法唤醒
